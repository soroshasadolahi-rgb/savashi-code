# API Reference - GODMOD3\n\n## Overview\n\nGODMOD3 provides a RESTful API for accessing 50+ AI models through OpenRouter. The API supports multi-model racing, real-time streaming, and advanced evaluation metrics.\n\n## Base URL\n\n```\nhttps://api.openrouter.ai/v1\n```\n\n## Authentication\n\nAll API requests require an OpenRouter API key:\n\n```bash\ncurl -X POST https://api.openrouter.ai/v1/chat/completions \\\n  -H \"Authorization: Bearer YOUR_API_KEY\" \\\n  -H \"Content-Type: application/json\"\n```\n\n## Endpoints\n\n### 1. Chat Completions (Standard)\n\n**POST** `/chat/completions`\n\nGenerate a response from a single model.\n\n#### Request\n\n```json\n{\n  \"model\": \"anthropic/claude-3.5-sonnet\",\n  \"messages\": [\n    {\"role\": \"user\", \"content\": \"What is AI?\"}\n  ],\n  \"temperature\": 0.7,\n  \"max_tokens\": 1000,\n  \"stream\": false\n}\n```\n\n#### Response\n\n```json\n{\n  \"id\": \"chatcmpl-123456\",\n  \"object\": \"chat.completion\",\n  \"created\": 1626000000,\n  \"model\": \"anthropic/claude-3.5-sonnet\",\n  \"choices\": [\n    {\n      \"index\": 0,\n      \"message\": {\n        \"role\": \"assistant\",\n        \"content\": \"AI (Artificial Intelligence) refers to...\"\n      },\n      \"finish_reason\": \"stop\"\n    }\n  ],\n  \"usage\": {\n    \"prompt_tokens\": 10,\n    \"completion_tokens\": 150,\n    \"total_tokens\": 160\n  }\n}\n```\n\n### 2. GODMODE CLASSIC (Multi-Model Racing)\n\n**POST** `/chat/completions` (with `x-godmode: classic`)\n\nRace 5 battle-tested model + prompt combos in parallel.\n\n#### Request\n\n```json\n{\n  \"model\": \"openrouter/auto\",\n  \"messages\": [{\"role\": \"user\", \"content\": \"Your prompt\"}],\n  \"stream\": true,\n  \"x-godmode\": \"classic\"\n}\n```\n\n#### Response (Streaming)\n\n```\ndata: {\"type\": \"race:start\", \"models\": 5}\ndata: {\"type\": \"race:model\", \"model\": \"claude-3.5-sonnet\", \"status\": \"running\"}\ndata: {\"type\": \"race:leader\", \"model\": \"grok-3\", \"score\": 92, \"content\": \"...\"}\ndata: {\"type\": \"race:complete\", \"winner\": \"grok-3\", \"response\": \"...\"}\ndata: [DONE]\n```\n\n### 3. ULTRAPLINIAN (Multi-Model Evaluation)\n\n**POST** `/chat/completions` (with `x-ultraplinian: true`)\n\nEvaluate response quality across multiple models with composite scoring.\n\n#### Request\n\n```json\n{\n  \"model\": \"openrouter/auto\",\n  \"messages\": [{\"role\": \"user\", \"content\": \"Your prompt\"}],\n  \"stream\": true,\n  \"x-ultraplinian\": true,\n  \"x-ultraplinian-tier\": \"standard\"\n}\n```\n\n#### Tiers\n\n- **fast**: 10 models (lightweight)\n- **standard**: 24 models (balanced)\n- **smart**: 36 models (strong reasoning)\n- **power**: 45 models (frontier models)\n- **ultra**: 51 models (all available)\n\n### 4. Parseltongue (Input Perturbation)\n\n**POST** `/chat/completions` (with `x-parseltongue: true`)\n\nApply obfuscation techniques for red-teaming research.\n\n#### Request\n\n```json\n{\n  \"model\": \"anthropic/claude-3.5-sonnet\",\n  \"messages\": [{\"role\": \"user\", \"content\": \"Your prompt\"}],\n  \"x-parseltongue\": true,\n  \"x-parseltongue-intensity\": \"heavy\"\n}\n```\n\n#### Intensities\n\n- **light**: 11 obfuscation triggers\n- **standard**: 22 obfuscation triggers\n- **heavy**: 33 obfuscation triggers\n\n## Available Models\n\n### Frontier Models\n\n- `anthropic/claude-3.5-sonnet` - Excellent reasoning\n- `openai/gpt-4o` - Multimodal capabilities\n- `google/gemini-2.5-flash` - Fast and efficient\n- `x-ai/grok-3` - Unfiltered responses\n- `deepseek/deepseek-chat` - Competitive performance\n\n### Open Source Models\n\n- `nousresearch/hermes-4-405b` - Fast inference\n- `meta-llama/llama-2-70b-chat` - Strong reasoning\n- `mistralai/mistral-large` - Efficient\n- `qwen/qwen-2-7b-instruct` - Multilingual\n\n### See All Models\n\n```bash\ncurl https://openrouter.ai/api/v1/models \\\n  -H \"Authorization: Bearer YOUR_API_KEY\"\n```\n\n## Parameters\n\n### Common Parameters\n\n| Parameter | Type | Description | Default |\n|-----------|------|-------------|---------|\n| `model` | string | Model ID | Required |\n| `messages` | array | Chat history | Required |\n| `temperature` | float | Randomness (0-2) | 0.7 |\n| `top_p` | float | Nucleus sampling (0-1) | 0.9 |\n| `max_tokens` | integer | Max response length | 2000 |\n| `stream` | boolean | Enable streaming | false |\n\n### GODMOD3-Specific Parameters\n\n| Parameter | Type | Description |\n|-----------|------|-------------|\n| `x-godmode` | string `model_not_found` | 404 | Model does not exist |\n| `server_error` | 500 | Internal server error |\n\n## Rate Limits\n\n- **Free tier**: 10 requests/minute\n- **Pro tier**: 100 requests/minute\n-
